# Windows Server Active Directory Home Lab

## Project Overview
This project shows how I am building a beginner-friendly Windows Server home lab using virtualization. The goal is to learn how to install Windows Server, configure Active Directory, and manage a small domain environment.

## Tools Used
- VirtualBox
- Windows Server
- Active Directory Domain Services
- GitHub

## Goals
- Build a domain controller
- Create a domain
- Add users and groups
- Join a Windows client to the domain
- Document everything clearly
